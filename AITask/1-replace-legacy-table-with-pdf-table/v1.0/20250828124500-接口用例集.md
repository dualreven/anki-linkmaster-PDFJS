### 2.1 addPDF 接口

#### 2.1.1 功能描述
添加PDF文件到表格中。

#### 2.1.2 请求参数
```json
{
  "filePath": "string"
}
```

#### 2.1.3 响应格式
成功响应：
```json
{
  "status": "success"
}
```

错误响应：
```json
{
  "error": "Invalid file path"
}
```

#### 2.1.4 正向用例
- 输入有效的文件路径，验证返回状态为"success"

- 输入无效的文件路径，验证返回400错误码
